# MY FIRST BLOG
A multiple Page React/node app that allows users to register/sign-in ands create posts , update their posts , view all posts .

## Setup

Install dependencies with `npm install`.

## Running Webpack Development Server

```sh
npm start
```
## Dependencies
  1.bcrypt
  2.express
  3.morgan
  4.multer
  5.mongoose
  6.axios
